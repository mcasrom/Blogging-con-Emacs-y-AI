Linux Script borra todos archivos y carpetas vacias

¡Hola a todos!

Anuncio aquí una nueva sección en mi blog dedicada a scripts útiles para Linux. En esta sección, compartiré scripts que he desarrollado o encontrado útiles en mi día a día.

Para empezar, he creado un script que busca y elimina todos los archivos y carpetas vacías (0 bytes) que tengan una antigüedad de ayer o más. Esto es útil para limpiar directorios y eliminar archivos basura.

Script en Bash: 

#!/bin/bash

# Encuentra y elimina archivos vacíos
find . -type f -size 0 -mtime +1 -delete

# Encuentra y elimina directorios vacíos
find . -type d -empty -mtime +1 -delete

echo "Tarea finalizada."

Explicación del Script:

  • `find . -type f -size 0 -mtime +1 -delete`: Busca archivos (`-type f`) en el directorio actual (`.`) que tengan un tamaño de 0 bytes (`-size 0`) y que hayan sido modificados hace más de un día (`-mtime +1`). Luego, los elimina (`-delete`).
  • `find . -type d -empty -mtime +1 -delete`: Busca directorios (`-type d`) en el directorio actual (`.`) que estén vacíos (`-empty`) y que hayan sido modificados hace más de un día (`-mtime +1`). Luego, los elimina (`-delete`).
  • `echo "Tarea finalizada."`: Imprime un mensaje indicando que la tarea ha finalizado.

¿Cómo usar el script?

  1. Copia el script y guárdalo en un archivo con extensión `.sh`, por ejemplo, `limpiarvacios.sh`.
  2. Dale permisos de ejecución al script: `chmod +x limpiarvacios.sh`.
  3. Ejecuta el script desde la terminal: `./limpiarvacios.sh`.

Advertencia:

  • Este script elimina archivos y directorios de forma irreversible. Úsalo con precaución y asegúrate de entender lo que hace antes de ejecutarlo.
  • Modifica el valor de `-mtime +1` si deseas cambiar la antigüedad de los archivos y directorios a eliminar.
